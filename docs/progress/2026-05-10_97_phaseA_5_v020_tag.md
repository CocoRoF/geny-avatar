## tag 부여

A.1~A.5 commit 모두 main 에 들어간 직후 annotated tag:

```bash
git tag -a v0.2.0 -m "v0.2.0 — Geny integration ready (Phase A complete)"
git push origin v0.2.0
```

Geny submodule 등록 시 이 tag 를 pin (B.1):

```bash
# in Geny repo:
git submodule add https://github.com/CocoRoF/geny-avatar vendor/geny-avatar
cd vendor/geny-avatar
git checkout v0.2.0
cd ../..
git add .gitmodules vendor/geny-avatar
git commit -m "Pin geny-avatar to v0.2.0"
```
