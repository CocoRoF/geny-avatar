## 디자인 결정

### Adapter는 store 밖

Adapter 인스턴스는 mutable class라 immer freeze에 걸림. 별도 ref/usePuppet 반환값으로 두고, store는 도메인 상태만. Page 단의 작은 wrapper 함수가 "store update + adapter 호출" 둘 다.

### Store 모양 (V1)

```ts
type EditorState = {
  avatar: Avatar | null
  selectedLayerIds: LayerId[]
  playingAnimation: string | null
  layerFilter: string
  visibilityOverrides: Record<LayerId, boolean>  // defaults에서 복사 후 사용자 편집
  // colorOverrides는 1.4b
  
  // actions
  setAvatar(a: Avatar | null): void
  setLayerVisibility(id, visible): void
  bulkSetLayerVisibility(ids, visible): void
  selectLayers(ids): void
  setPlaying(name): void
  setLayerFilter(q): void
  reset(): void
}
```

향후(1.4b 이후): `colorOverrides`, `aiJobs`, `viewport`, `panelMode`, `history`.

### Selector 단위 subscribe

작은 컴포넌트가 자기 필요한 slice만 selector로 가져와 re-render 최소화:
```ts
const layers = useEditorStore((s) => s.avatar?.layers ?? [])
const visibility = useEditorStore((s) => s.visibilityOverrides)
```
