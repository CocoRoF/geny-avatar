## 검증 절차 (매 Stage 공통)

1. `pnpm typecheck && pnpm lint` — 커밋 게이트.
2. 수동 스모크 (빌트인 Hiyori + spineboy):
   업로드/빌트인 열기 → 레이어 토글 → Decompose 3모드 저장/재진입 → Gen 1회 →
   apply → 라이브 반영 → Save File → 재import → Export Model → 라이브러리 복귀.
3. Stage 4 이후: 페이지 오버라이드 적용 → 레이어 오버라이드 공존 → revert →
   export byte-비교(오버라이드 없을 때 기존과 동일).
4. 회귀 의심 시 해당 결함 ID 의 재현 절차를 progress 로그에 기록해 두고 재확인.
