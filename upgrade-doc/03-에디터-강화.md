# 03 — 에디터 강화 (G2: "간단하지만 확실한" 포토샵류 텍스처 에디터)

전제: Stage 2 의 결함 수정(E1–E14)이 끝난 뒤 시작한다. 기존 Mask/Split 모드는
건드리지 않고 **Paint 모드를 중심으로** 확장한다.

## 현재 Paint 모드의 좋은 점 (보존)

- StrokeEngine: spacing 보간 + pen pressure + hardness gradient — 웹 에디터 평균 이상.
- 트라이앵글 풋프린트 클립: 칠한 픽셀이 레이어 영역 밖으로 절대 새지 않음.
- 저장 경로: PNG → `layerTextureOverrides` → 라이브 아틀라스 반영. 이 파이프는 유지.

## 강화 항목 (우선순위순)

### P-1. 브러시 불투명도 + 직선
- OptionsBar 에 Opacity 슬라이더(1–100%) 추가. StrokeEngine dab 알파에 곱.
- Shift+클릭 = 마지막 점→클릭 점 직선 스트로크 (포토샵 표준). StrokeEngine 에
  `lineTo(from, to)` 한 번의 보간 패스로 구현.

### P-2. 색 워크플로
- 전경/배경 색 (X 스왑, D 리셋) — 현재 X 는 brush↔eraser 토글이므로 **brush↔eraser 는
  E/B 로 유지하고 X 를 색 스왑으로 재배정** (기존 단축키 안내 갱신 포함).
- 최근 색 스와치 8칸 (스포이드/픽커 선택 시 push, localStorage 영속).
- BrushCursor 링이 전경색을 미리보기 (현재 항상 accent 색).

### P-3. 블렌드 모드 (소박하게)
- 브러시 합성 모드 select: normal / multiply / screen / overlay — Canvas2D
  `globalCompositeOperation` 으로 끝나는 범위만. 레이어 시스템은 만들지 않는다 (스코프 제외).

### P-4. 선택 영역과 Paint 의 결합
- 이미 있는 wand 선택을 paint 모드에서도 사용: 선택 활성 시 브러시/버킷이 선택 내부로
  클립 (Path2D 클립 합성). Ctrl+D 해제, Ctrl+Shift+I 반전 추가.

### P-5. 조정 (선택 범위 한정)
- Hue/Saturation/Lightness 슬라이더 1조 — 선택(없으면 전체)에 per-pixel HSL shift,
  worker 에서 처리 (기존 floodFillWorker 패턴 재사용). 미리보기는 컴포지터에 임시 캔버스로.
- 이것이 docs-upgrade Phase 2 "tint fast-path" 의 에디터판 최소 구현이다 — drawable
  multiplyColor 방식은 G3 이후 별도.

### P-6. 전체 아틀라스 페이지 편집 (G3 와 공유되는 기반)
- 새 진입점: LayersPanel 상단 "페이지 편집" — layer 단위가 아니라 **아틀라스 페이지
  전체**를 DecomposeStudio paint 모드로 연다.
- 저장 대상은 Stage 4 에서 만드는 `pageTextureOverrides[pageIndex]`.
- 레이어 풋프린트 클립 없음(페이지 전체가 캔버스). 안내선으로 각 레이어 bbox 외곽선
  토글 표시 (이미 있는 triangles 데이터 재사용).

## 비목표 (의식적으로 안 함)

- 레이어 시스템, smudge/clone/blur 브러시, 텍스트 도구, 변형(transform) 도구.
- 모바일/터치 — 기존 방침 유지.
